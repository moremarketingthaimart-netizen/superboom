# Skill: daily-digest

Pull together everything that happened in the repo — commits, PRs, issues — into a clean digest ready to share with your team.

## Arguments

`$ARGUMENTS` — time range and format (e.g. "today", "this week", "last 7 days", "format=slack", "format=markdown")

## Steps

### Step 1 — Determine time range

Parse `$ARGUMENTS` for time range:
- "today" → `--since="midnight"`
- "this week" / "7 days" → `--since="7 days ago"`
- "yesterday" → `--since="yesterday midnight" --until="midnight"`
- Default (no arg) → today

### Step 2 — Gather git data

```bash
# Commits in range
git log --since="midnight" --oneline --no-merges --format="%h %s (%an)" 2>/dev/null

# Files most changed today
git log --since="midnight" --name-only --pretty=format: 2>/dev/null | sort | uniq -c | sort -rn | head -10

# Authors active today
git log --since="midnight" --format="%an" 2>/dev/null | sort | uniq -c | sort -rn
```

### Step 3 — Gather GitHub data (if available)

```bash
# Open PRs
gh pr list --state open --json number,title,author,createdAt 2>/dev/null | head -10

# Recently merged PRs
gh pr list --state merged --json number,title,author,mergedAt --search "merged:>$(date -d '1 day ago' '+%Y-%m-%d')" 2>/dev/null

# Open issues updated today
gh issue list --state open --json number,title,labels,updatedAt 2>/dev/null | head -10
```

Skip GitHub steps if `gh` is not authenticated.

### Step 4 — Format the digest

**Slack format** (when `format=slack` in `$ARGUMENTS`):

```
:rocket: *Daily Dev Digest — <date>*

*Commits today (<N>):*
• `abc1234` Fix auth token expiry bug — Alice
• `def5678` Add user profile endpoint — Bob

*PRs merged:*
• #42 — Implement dark mode (merged by Carol)

*Open PRs needing review (<N>):*
• #44 — Database connection pooling
• #45 — Fix mobile layout

*Top changed files:*
• `src/api/auth.ts` (5 commits)
• `src/components/Profile.tsx` (3 commits)

_Generated by superboom/daily-digest_
```

**Markdown format** (default):

```markdown
# Daily Dev Digest — <date>

## Commits (<N> today)
| Hash | Message | Author |
|------|---------|--------|
| `abc1234` | Fix auth token expiry | Alice |

## Pull Requests
### Merged Today
- #42 Implement dark mode (Carol)

### Open / Awaiting Review  
- #44 Database connection pooling
- #45 Fix mobile layout

## Active Contributors
- Alice: 5 commits
- Bob: 3 commits

## Most Changed Files
1. `src/api/auth.ts` — 5 commits
2. `src/components/Profile.tsx` — 3 commits
```

### Step 5 — Output

Print the formatted digest. If no commits found, output:
```
No commits in the selected time range. The codebase is at rest. ☕
```

Offer to save as a file: `digest-<date>.md`
